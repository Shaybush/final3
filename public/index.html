Hey! I'm Shay Bushary and this is an api of books I've written.

Books Routes

Get information about books via a full API

Method URL Description
Get /books?page=number Get a list of all the books , present 10 results by page number
Get /books?perPage=number Get X books in each page
Get /books?reverse=sort-key Reverse by sort key: yes/no
Get /books?sort=key-word From which value to sort by
Get /books/price?min=number&max=number Price range
Get /books/category/:category-name Get all books from the same category
Get /books/search?s=keyword Search products by name or info

Books Actions

Add, remove and edit allowed only for registered users

Method URL Body Description
Post /books
  1. name (required)
  2. author (required)
  3. type (required)
  4. info (required)
  5. category (required)
  6. img_url
  7. price (required)
 Add a new Book Must to be connected
Edit /books/:id-key
  1. name (required)
  2. author (required)
  3. type (required)
  4. info (required)
  5. category (required)
  6. img_url
  7. price (required)
 Edit a new Book Must to be connected
Delete /books/:id-key Delete a new Book Must to be connected

Get /users?perPage=number Get X users in each page
Get /users?reverse=sort-key Reverse by sort key: yes/no
Get /users?sort=key-word From which value to sort by
